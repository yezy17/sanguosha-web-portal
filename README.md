# Sanguosha Web Portal

高可玩性、易分享的三国杀联机平台（Web 门户）——采用“成熟游戏引擎 + 自建 Web 门户”架构。

## 目标
- 可联机：稳定开房/加入/开局
- 易分享：邀请链接一键拉人
- 高可玩：规则正确、断线可恢复、对局可回放
- 可迭代：自动化测试与回归，减少人肉测

## 架构概览
- `apps/web`: 门户前端（大厅、房间、邀请、对局壳）
- `apps/gateway`: 门户后端（认证、房间、会话、引擎适配）
- `packages/shared`: 共享类型、协议、校验
- `docs/`: 架构、路线图、测试策略

详见 `docs/architecture.md`。

## 开发原则
1. 服务端权威状态，前端只渲染。
2. 每局必须有事件日志，可 replay 复现。
3. 每个可运行里程碑立即 push。
